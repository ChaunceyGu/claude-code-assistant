# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-03-28

### Added
- Initial release of Claude Code Desktop Assistant
- Dashboard with 6 feature cards:
  - Permission Management (🛡️)
  - Recent Projects (📂)
  - Quick Commands (⚡)
  - Config Editor (⚙️)
  - Config Profiles (🔄)
  - Skill Management (🧩)
- Tauri-based desktop application (Rust + Web)
- Responsive UI with animations
- Toast notification system
- GitHub Actions CI/CD for automated builds

### Technical
- Rust backend with Tauri framework
- Vanilla JavaScript frontend (no frameworks)
- CSS3 with CSS variables for theming
- GitHub Actions workflow for Windows builds
